# 🌿 Mente Plena - Clínica de Psicologia

![Banner do Projeto](./public/screenshots/banner.png)

Uma Landing Page moderna, minimalista e acolhedora desenvolvida para a clínica **Mente Plena**. O projeto foca em proporcionar uma experiência de usuário tranquila, refletindo o cuidado e o acolhimento do atendimento psicológico.

## ✨ Demonstração
🔗 [Visualizar Projeto Online](https://mente-plena.vercel.app) 

---

## 🚀 Tecnologias Utilizadas

O projeto foi construído utilizando as melhores práticas de desenvolvimento web moderno:

* **React 18**: Biblioteca principal para construção da interface.
* **Vite**: Tooling de próxima geração para um desenvolvimento rápido.
* **Tailwind CSS**: Framework utilitário para estilização responsiva e customizada.
* **Lucide React / SVGs**: Ícones leves e elegantes.
* **Intersection Observer API**: Utilizada para as animações de "Reveal" (surgimento) ao scroll.

---

## 🎨 Identidade Visual

A paleta de cores foi cuidadosamente selecionada para transmitir calma e profissionalismo:
* **Sage (Verde Folha)**: Representa crescimento e equilíbrio.
* **Cream (Creme)**: Traz suavidade e conforto visual.
* **Warm (Terracota)**: Pontos de destaque que remetem ao acolhimento humano.

**Tipografia:**
* **Display**: *Cormorant Garamond* (Elegância e seriedade).
* **Body**: *DM Sans* (Leitura clara e moderna).

---

## 🛠️ Funcionalidades

- [x] **Layout Responsivo**: Adaptado para Desktop, Tablet e Mobile.
- [x] **Navegação Inteligente**: Navbar que muda de estilo ao rolar a página.
- [x] **Seção de Serviços**: Grid interativo apresentando as especialidades.
- [x] **Horário Dinâmico**: O card de "Próximo Horário" no Hero atualiza automaticamente conforme a hora atual do usuário.
- [x] **Animações On-Scroll**: Elementos que surgem suavemente conforme o usuário navega.
- [x] **Integração com Google Maps**: Localização real da clínica no rodapé.

## 💻 Como rodar o projeto localmente

1. **Clone o repositório:**
   ```bash
   git clone [https://github.com/seu-usuario/mente-plena.git](https://github.com/seu-usuario/mente-plena.git)
   ```

2. **Entre na pasta do projeto:**
   ```bash
   cd mente-plena
   ```

3. **Instale as dependências:**
   ```bash
   npm install
   ```
4. **Inicie o servidor**
   ```bash
   cd mente-plena
   ```
